How to use PRISA

The Personal Re-Identification System software Application (PRISA) is a graphical user interface between the user and the Personal Re-Identification System (PRIS) software created at the Missouri State University. It has the ability to recognize the features of an individual across different media through algorithm training. Currently PRIS is the most accurate identification software known.


To use PRISA, go to the upload page. Find and click on the yellow button labelled "Add Media" to open a File Explorer window. Navigate through File Explorer to the destination of the stored media. Choose the image or video you wish to upload by clicking on it. To upload the image select "Open" in the bottom right of the window. You may cancel your upload any time by clicking on "Cancel" in the lower right corner of the window. You have finished the uploading process.


Once you are done uploading your media PRIS will begin processing your image or video file. Once it is done a green box will display at the bottom of the uploads page to alert you. The results of the 3 closest matching images in the PRIS database will appear beside the uploaded image and ask, "Is this the same person? Click on image(s) to confirm." Click on all images that display the same person that appears in the originally uploaded media. This will create a profile for the person of interest (POI). The profiles in PRISA will associate data within PRIS to a specific person.
